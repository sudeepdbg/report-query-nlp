# report-query-nlp


### Supporting queries:

Question	Expected Result
"Show me all delayed work orders"	List of work orders with status 'Delayed'
"List at risk work orders"	Same as above
"Get delayed work orders"	Same as above
"What are the delayed work orders?"	Same as above
"Show MAX Australia content"	List of content planned for MAX Australia
"List content for MAX Australia"	Same as above
"Get MAX US content"	List for MAX US
"Show MAX US items"	Same as above
"List top vendors"	Vendors ranked by work order count (descending)
"Show vendor a work orders"	(Note: "vendor a" triggers the top vendors query, which groups all vendors)
"Show active deals"	List of deals with status 'Active'
"List pending approval deals"	Deals with status 'Pending'
"Get in progress work orders"	Work orders with status 'In Progress'
"Show not ready content"	Content with status 'Not Ready'
